## 1. InnoDB 索引结构？

**A:** B+ 树。

## 2. 聚簇 vs 二级？

**A:** 聚簇叶子=整行；二级叶子=主键，可能回表。

## 3. 覆盖索引？

**A:** 查询列都在二级索引，免回表。

## 4. 最左前缀？

**A:** (a,b,c) 从 a 起连续使用。

## 5. 索引失效常见？

**A:** 函数包列、左模糊、隐式转换、违反最左。

## 6. 默认隔离级别？

**A:** RR 可重复读。

## 7. 脏读/不可重复/幻读？

**A:** 未提交 / 行被改 / 范围行数变。

## 8. MVCC 一句话？

**A:** 版本链+ReadView，快照读少加锁。

## 9. 快照读 vs 当前读？

**A:** 普通 SELECT vs FOR UPDATE 等。

## 10. 慢 SQL 四步？

**A:** 慢日志→EXPLAIN→改索引/SQL→验证。

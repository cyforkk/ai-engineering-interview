## 为什么必须有金标集

LLM 输出非确定性；改 prompt「手感变好」可能在别的 case 静默变差。固定集 + rubric 才能回归。

## 离线与在线

离线可控可对比；在线反映真实分布与漂移。两者互补。

## 指标

正确/部分正确、拒答是否合适、引用是否可追溯、延迟与 token 成本——质量与商业一起看。

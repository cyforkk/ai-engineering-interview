# 问题记录：夜间模式标题 / 加粗发灰

## 现象

夜间模式下标题、`**加粗**` 文字偏灰，对比度差、像未激活状态。

## 原因

1. Docsify `vue.css` 给 `h1–h6` / `strong` 写了浅色主题用的深灰字色（如 `#2c3e50`），暗色下未全部覆盖会发脏发灰。  
2. 自定义样式里把 **blockquote**、**侧栏 strong 分组标题** 设成了 `--text-muted`，夜间更灰。  

## 修复

`index.html`：

- 增加 `--text-heading` / `--text-strong`（暗色接近白）  
- 强制 `h1–h6`、`strong/b`、`th`、侧栏分组标题用高对比色  
- blockquote 改回正文色，内部 strong 同步  

## 验证

本地打开站点 → 切 🌙 夜间 → 看 h2、表格表头、正文加粗、引用块加粗、侧栏「① 路径」等是否清晰偏白。

## P0 答法骨架

### 进程 vs 线程

- 进程：资源分配、地址空间隔离  
- 线程：调度单位、共享堆要同步  
- Java：线程共享堆 → 要线程安全  

### 用户态 / 内核态

- 用户不能随便碰硬件 → 系统调用进内核  
- 切换有成本 → 缓冲、批量、零拷贝  

### 上下文切换

- 保存/恢复寄存器；过多则 CPU 高吞吐差  
- 线程池、事件驱动减少无效切换  

### epoll（P1 但后端常问）

- 一线程听多连接；select/poll 线性扫  
- epoll 就绪队列，海量连接  
- Nginx/Redis/Netty  

### 死锁

- 四条件；破循环等待；jstack  

### 虚拟内存

- 页表映射；缺页慢；JVM 在虚拟地址空间

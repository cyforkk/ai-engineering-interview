## 二、P0 必会

### 2.1 进程 vs 线程

| | 进程 | 线程 |
|--|------|------|
| 地址空间 | 独立 | 共享 |
| 开销 | 大 | 小 |
| 通信 | IPC | 共享+同步 |

**高频题：** 区别？为何要线程？上下文切换代价？

### 2.2 用户态 / 内核态

特权不同；系统调用陷入内核；频繁切换影响性能。

## 三、P1 常考

### 3.1 IO 多路复用

| | 特点 |
|--|------|
| select | fd 有限，轮询 |
| poll | 无 1024 限制，仍轮询 |
| **epoll** | 事件驱动，海量连接（Linux） |

**场景：** Nginx、Redis、Netty。

### 3.2 死锁

互斥、占有等待、不可抢占、循环等待。  
避免：顺序加锁、超时。

### 3.3 虚拟内存

进程虚拟地址 → 物理页；缺页中断；与 JVM 堆同属进程地址空间。

### 3.4 同步异步 / 阻塞非阻塞

概念会区分即可；结合 NIO 说。

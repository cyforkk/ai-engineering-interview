## 1. #{} 与 ${} 区别？（必考）

| | `#{}` | `${}` |
|--|-------|-------|
| 方式 | 预编译占位 `?` | 字符串拼接 |
| SQL 注入 | **安全** | **有风险** |
| 用途 | 参数值 | 表名/列名等动态标识（需白名单） |

**结论：能用 #{} 绝不用 ${}。**

---

## 2. 一级缓存？二级缓存？

| | 一级 | 二级 |
|--|------|------|
| 范围 | **SqlSession** | **Mapper 命名空间**（跨 Session，需配置） |
| 默认 | 开启 | 默认关，需显式开 |
| 注意 | Session 关闭/提交等会清 | 分布式下本地二级缓存易脏，慎用 |

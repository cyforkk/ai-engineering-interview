## 一、考察什么

| 维度 | 例子 |
|------|------|
| 动机 | 为何做这行/为何来本公司 |
| 协作 | 冲突、沟通、跨团队 |
| 抗压 | 加班、线上故障心态 |
| 成长 | 学习能力、反思 |
| 稳定性 | 职业规划、离职原因 |
| 价值观 | 诚信、客户意识 |

**原则：** 真实、具体、不攻击前司；用 **STAR** 讲事，用 **价值观** 收尾。

---

## 二、高频题清单（P0）

1. 自我介绍（1～2 分钟，偏经历动机）  
2. 为什么离开上家 / 为什么找工作？  
3. 为什么选我们公司/这个岗位？  
4. 职业规划 3～5 年？  
5. 优缺点？  
6. 最有成就感的事？  
7. 失败/后悔的事？  
8. 和同事/领导意见冲突？  
9. 被质疑/被怼怎么处理？  
10. 如何学习新技术？  
11. 如何看待加班？  
12. 期望薪资？  
13. 还有别的 offer 吗？  
14. 你有什么问我的？  
15. 团队角色：主力还是打杂？  

---

## 三、答题要点速查

| 题 | 要点 |
|----|------|
| 离职 | 发展空间/业务方向/家庭，**不说**公司烂、领导差 |
| 来本公司 | 业务×技术×阶段匹配，各 1 句真实点 |
| 规划 | 先称职再专家/带小组，贴岗位 |
| 缺点 | 真缺点+已改进，勿假缺点「太努力」 |
| 冲突 | 对事不对人、数据对齐、升级机制 |
| 加班 | 接受业务节奏，也谈效率与健康边界 |
| 薪资 | 区间+看总包与成长，勿漫天 |

---

## 四、自我介绍结构（非纯技术）

```text
谁：背景与现状
做过什么：1～2 段经历含结果
为什么合适：技能匹配岗位
动机：为何看这个机会
```

---

## 五、红线

- 撒谎可验证经历  
- 贬低前同事  
- 只谈钱无业务兴趣  
- 完全无缺点  
- 对业务零了解还硬吹

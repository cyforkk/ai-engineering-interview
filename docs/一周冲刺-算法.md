# 一周冲刺包 · 算法（笔试 / 手写）

> **只碰本页列出的题与文档**，别从 Hot100 全文乱刷。  
> 每天约 2～3 小时：模板 30 分钟 + 做题 90 分钟 + 口述/复盘 30 分钟。

**入口：** [算法入口.md](./算法入口.md)  
**配套：** [进度清单](./进度清单.md) · [T0 错因](./算法T0错因与口述.md) · [频率总榜](./算法面试频率总榜.md) · [Hot100 分类](./算法Hot100频率分类.md) · [默写代码册](./算法高频默写代码册.md) · [面渣](./算法面渣级口述.md)

**目标：** 总榜 **T0 全会讲 + 能默写**；T1 会模板；代码册对照过一遍。

---

## 每日规则

```text
① 先打开分类文对应章节，抄/默「模板」
② 按本日题号写（Java 或 Python 二选一，全周统一）
③ 每题必须说：暴力 → 优化 → 复杂度 → 边界
④ 卡壳 >15 分钟先看模板再自己重写，禁止抄完就过
```

## D7 自测（遮答案）

对每题计时 15～25 分钟（含口述）：

| # | 题 | 自检 |
|---|-----|------|
| 1 | 两数之和 | 哈希一次扫描 |
| 2 | 反转链表 | 三指针 / 递归边界 |
| 3 | 无重复最长子串 | 窗口 + 字典 |
| 4 | 层序遍历 | 队列 BFS |
| 5 | LRU | HashMap + 双向链表 |
| 6 | 三数之和 | 排序 + 对撞 + 去重 |
| 7 | 岛屿数量 | DFS/BFS 沉岛 |
| 8 | 爬楼梯 / 最大子数组 | DP 状态转移一句话 |
| 9 | 合并区间 | 排序 + 扫 |
| 10 | 全排列 | 回溯模板 |

**及格：** 10 题中 ≥7 题能独立写出可运行骨架并讲清复杂度。

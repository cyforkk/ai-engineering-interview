## 一、优先级总览

| 级 | 题型 | 频率 |
|----|------|------|
| **P0** | 数组双指针、链表、二分、哈希、栈、二叉树遍历/层序 | 极高 |
| **P0** | 排序思想、TopK/堆、滑动窗口、字符串 | 极高 |
| **P1** | DFS/BFS、回溯入门、前缀和、单调栈 | 高 |
| **P2** | 动态规划经典、并查集、字典树、图最短路 | 中（看公司） |

**面试算法目标：** 说清思路 → 写对边界 → 说复杂度；不是刷完所有题。

## 二、复杂度与答题模板（P0）

### 2.1 复杂度

| 记号 | 含义 |
|------|------|
| O(1) | 哈希取值、数组下标 |
| O(log n) | 二分、平衡树高 |
| O(n) | 单次遍历 |
| O(n log n) | 快排/归并平均或保证 |
| O(n²) | 双层循环 |

空间：是否原地、递归栈、额外哈希。

### 2.2 白板模板

```text
1. 复述题意 + 例子
2. 暴力思路与复杂度
3. 优化思路（数据结构/不变量）
4. 写代码（先主路径后边界）
5. 自测：空、单元素、极端
6. 时间/空间复杂度
```

## 三、P0 题型与代表题

### 3.1 数组 / 双指针

| 题 | 思路 |
|----|------|
| 两数之和 | 哈希 val→index |
| 三数之和 | 排序+双指针去重 |
| 盛水最多 | 左右指针贪心 |
| 移动零 | 快慢指针 |
| 有序数组合并 | 双指针从后往前 |

### 3.2 链表

| 题 | 思路 |
|----|------|
| 反转链表 | 迭代三指针 / 递归 |
| 环检测 | 快慢指针 |
| 相交链表 | 双指针换头 |
| 删除倒数第 n | 前后指针 |
| 合并有序链表 | 哑头 |

### 3.3 栈 / 队列

| 题 | 思路 |
|----|------|
| 有效括号 | 栈匹配 |
| 最小栈 | 辅助栈 |
| 用栈实现队列 | 双栈 |

### 3.4 二分

| 题 | 思路 |
|----|------|
| 搜索插入位置 | 标准二分 |
| 旋转数组找最小/目标 | 有序半边判断 |
| 搜索二维矩阵 | 从角或二分 |

**边界：** `left/right` 开闭、`mid` 防溢出、`while` 条件。

### 3.5 哈希

| 题 | 思路 |
|----|------|
| 两数之和 | map |
| 字母异位词 | 计数 |
| 最长连续序列 | set O(n) |

### 3.6 树

| 题 | 思路 |
|----|------|
| 前中后序 | 递归/迭代 |
| 层序 | 队列 |
| 最大深度 | 递归 |
| 对称/相同树 | 递归 |
| 最近公共祖先 | 递归分治 |
| BST 验证/第K小 | 中序 |

### 3.7 排序与 TopK

| 点 | 结论 |
|----|------|
| 快排 | 平均 n log n，最坏 n²，不稳定 |
| 归并 | n log n，稳，费空间 |
| 堆排 | n log n |
| TopK | 大小为 K 的堆；或快排划分 |

### 3.8 滑动窗口 / 字符串

| 题 | 思路 |
|----|------|
| 无重复最长子串 | 窗口+哈希 |
| 最小覆盖子串 | 窗口+计数（稍难） |
| 反转字符串/词 | 双指针 |
| 最长回文子串 | 中心扩展 / DP |

## 四、P1 加码

| 类型 | 代表 | 要点 |
|------|------|------|
| BFS | 二叉树层序、最短步数网格 | 队列 |
| DFS/回溯 | 全排列、组合总和 | 路径+撤销 |
| 前缀和 | 子数组和为 K | preSum+哈希 |
| 单调栈 | 每日温度、柱状图最大矩形 | 维护单调 |

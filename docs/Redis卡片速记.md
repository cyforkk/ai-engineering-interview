## 1. 为何快？

**A:** 内存、命令单线程、高效结构、IO 多路复用。

## 2. 五类型场景？

**A:** String 缓存/锁；Hash 对象；List 列表；Set 去重；ZSet 排行。

## 3. 穿透？

**A:** 查不存在→空值/布隆/校验。

## 4. 击穿？

**A:** 热点过期→互斥重建/逻辑过期。

## 5. 雪崩？

**A:** 大量同时过期或挂→TTL 抖动+HA+限流。

## 6. 分布式锁命令？

**A:** SET key val NX EX。

## 7. 删锁为何 Lua？

**A:** 校验 value 再 del 原子，防误删。

## 8. Cache Aside 写？

**A:** 先 DB 再删缓存。

## 9. 持久化？

**A:** RDB 快照；AOF 日志；可混合。

## 10. 缓存雪崩和击穿差？

**A:** 面 vs 点（热点单 key）。

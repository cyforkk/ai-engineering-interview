# 模拟面试 · 行为 + STAR 15 问

> 用法：遮住「应覆盖要点」，**用自己的真实经历**填空口述（2～3 分钟/题）。  
> 配套：[行为面渣](./行为面试面渣级口述.md) · [项目映射模板](./项目映射模板.md) · [STAR·Java](./项目STAR范例-Java.md) · [STAR·AI](./项目STAR范例-AI.md)

**及格线：** 15 题中至少 **11** 题结构完整（有情境、行动、结果，不抱怨前公司）。

---

## 开场与动机

### 1. 请做 90 秒自我介绍

**应覆盖：** 身份年限 → 方向 → 1 个结果 → 3 个栈 → 为何匹配岗位。  
**链接：** [行为面渣](./行为面试面渣级口述.md)

### 2. 为什么看新机会 / 为什么离职？

**应覆盖：** 阶段目标完成；正向动机；不抱怨不泄密。  

### 3. 为什么选择我们公司 / 这个岗位？

**应覆盖：** 业务 + 技术 + 阶段三点；最好有一个具体产品/技术点。  

### 4. 职业规划 1 年 / 3～5 年？

**应覆盖：** 先胜任 → 独立模块 → 深度或带人；务实可验证。  

---

## 项目 STAR（核心）

### 5. 挑一个最能代表你的项目，按 STAR 讲 3 分钟

**应覆盖：** S 背景规模；T 你的任务；A **你个人**行动与取舍；R 指标。  
**链接：** [项目映射](./项目映射模板.md)

### 6. 项目中最大的技术难点是什么？你怎么权衡方案？

**应覆盖：** 备选方案；取舍维度（成本/复杂度/风险）；落地结果。  

### 7. 有没有做砸 / 回滚 / 事故？你怎么处理与复盘？

**应覆盖：** 止血 → 根因 → 防再发；不甩锅；学到什么。  

### 8. 你在项目里的贡献，如果去掉你会怎样？（验证含金量）

**应覆盖：** 可替换性诚实说；强调独特决策或模块；协作边界。  

### 9. 跨团队冲突或需求反复，举一例

**应覆盖：** 对齐目标/指标；数据或风险沟通；结果与关系。  
**链接：** [行为面渣·冲突](./行为面试面渣级口述.md)

---

## 协作、压力与学习

### 10. 讲一次 deadline 很紧你怎么排期

**应覆盖：** 砍范围；并行；风险提前暴露；质量底线。  

### 11. 优缺点各一个，缺点要有改进证据

**应覆盖：** 优点带结果；缺点具体 + 已改动作 + 可验证。  

### 12. 最近一次主动学习并用于工作的技术？

**应覆盖：** 动机 → 怎么学 → 用在哪 → 收益。  

### 13. 你怎么做 Code Review / 带新人（有则讲，无则讲协作）

**应覆盖：** 标准、反馈方式、一次具体改进。  

---

## 收尾高频

### 14. 期望薪资 / 到岗时间 / 有无其他 offer

**应覆盖：** 区间有依据；到岗诚实；offer 不夸张不贬低他司。  

### 15. 你有什么想问我的？

**应覆盖：** 2～3 个好问题（业务指标、团队分工、技术债、成功画像）；避免只问假期。  

---

## 自测打分（每题 0/1/2）

| 分 | 含义 |
|----|------|
| 0 | 空洞 / 抱怨 / 无结构 |
| 1 | 有事但缺结果或个人贡献不清 |
| 2 | STAR 完整，可追问 2 层 |

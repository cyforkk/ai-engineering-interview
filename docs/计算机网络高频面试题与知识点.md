## 1. TCP 和 UDP

- **TCP：** 打电话，接通才说，保证送到、顺序对——稳但手续多  
- **UDP：** 发短信广播，快，不保证对方一定收到——音视频常用  

不是 UDP 差，是场景不同。

---

## 2. 三次握手（为啥三次？）

两人通话前确认：我能说你能听、你能说我能听。  
三次比两次更能避免「迟到的旧呼叫」造成误会。  
面试能讲清「同步序号 + 双向确认」就很好。

---

## 3. 挥手和 TIME_WAIT

挂电话两边都要说再见（全双工）。  
主动挂的一方多等一会儿（TIME_WAIT），避免最后的再见丢了、旧包搅和下一次通话。  
CLOSE_WAIT 很多：多半是**程序忘了关连接**。

---

## 4. HTTPS

HTTP 裸奔；HTTPS 加了锁和证件（TLS + 证书）。  
大白话：证明你连的是真网站，路上内容加密。

## 5. 状态码速记

- 401：你是谁？（没登录）  
- 403：知道你是谁，但没权限  
- 502：网关后面的服务坏了  
- 504：网关后面的服务太慢超时

## 二、P0 必会

### 2.1 TCP vs UDP

| | TCP | UDP |
|--|-----|-----|
| 连接 | 有 | 无 |
| 可靠 | 是 | 否 |
| 场景 | HTTP、MySQL | DNS、音视频 |

### 2.2 三次握手

```text
SYN → SYN+ACK → ACK
```

**为何三次：** 确认双方收发；防历史 SYN。

### 2.3 四次挥手

```text
FIN → ACK → FIN → ACK（主动方 TIME_WAIT）
```

**TIME_WAIT：** 保证最后 ACK；等旧包消失（2MSL）。

**高频题：** 握手/挥手过程？为何三次不是两次？TIME_WAIT？

---

### 2.4 HTTP / HTTPS

| 点 | 结论 |
|----|------|
| HTTPS | HTTP + TLS；证书 + 非对称换钥 + 对称通信 |
| GET/POST | 语义读/写；幂等与缓存习惯不同 |
| 状态码 | 200、301/302、304、401/403/404、500/502/504 |

**高频题：** HTTP vs HTTPS？TLS 干什么？常见状态码？

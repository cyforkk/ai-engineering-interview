## P0 答法骨架（展开全文见面渣）

### TCP vs UDP

- TCP：有连接、可靠、有序，适合 HTTP/DB  
- UDP：无连接、快，适合 DNS/音视频  
- **怎么说：** 先对比再讲场景选型，别贬 UDP  

### 三次握手

- SYN → SYN+ACK → ACK  
- **为何三次：** 防历史 SYN 误建连；确认双向通信  
- 面渣有完整故事版  

### 四次挥手 + TIME_WAIT

- 全双工分别关 → 四次  
- TIME_WAIT 2MSL：保最后 ACK；清旧包  
- CLOSE_WAIT 多：应用没 close  

### HTTPS

- HTTP 明文；HTTPS = HTTP + TLS  
- 证书证明身份；非对称换钥 + 对称传数据  

### 状态码

- 401 未认证 / 403 无权限  
- 502 上游坏 / 504 上游超时  

### 粘包

- TCP 字节流无边界 → 长度头/分隔符/定长

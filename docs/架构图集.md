## 1. AI 应用 / RAG 总览

```mermaid
flowchart LR
  U[客户端] --> G[API 网关<br/>鉴权/限流]
  G --> S[应用服务]
  S --> C{缓存}
  C -->|未命中| R[检索服务]
  R --> V[(向量库)]
  R --> E[(ES/关键词可选)]
  S --> L[LLM Provider]
  S --> DB[(业务 DB)]
  S --> O[日志/Trace/评测]
```

**口述一句：** 网关进服务，检索增强后调模型，全链路可观测。

## 2. Agent 循环

```mermaid
flowchart TD
  Q[用户目标] --> T[Thought]
  T --> A[Action 选工具]
  A --> X[执行工具]
  X --> O[Observation]
  O --> T
  T --> F[Final Answer]
```

**约束：** max_steps / 权限 / HITL 画在 Action 旁。

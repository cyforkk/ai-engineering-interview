## 1. 默认参数陷阱？

**A:** 可变默认只创建一次；用 None。

## 2. 浅拷贝 vs 深拷贝？

**A:** 外层 vs 递归。

## 3. 装饰器？

**A:** 函数包函数；wraps 保元数据。

## 4. 生成器？

**A:** yield 惰性，省内存。

## 5. with？

**A:** 上下文，保证清理。

## 6. GIL？

**A:** 同进程同时多一线程执行字节码。

## 7. CPU 密集？

**A:** 多进程/外部服务。

## 8. IO 密集？

**A:** 线程或 asyncio。

## 9. asyncio 禁忌？

**A:** 协程里阻塞调用堵事件循环。

## 10. FastAPI 适合？

**A:** async API、类型、OpenAPI；仍要超时限流。

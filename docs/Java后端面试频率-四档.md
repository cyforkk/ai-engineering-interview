## 高效背八股技巧

1. 先搞懂 **为什么**，再记 **怎么做**  
2. 每个核心点准备 **1～2 个项目场景**  
3. **画图**记忆（HashMap、线程池、JVM 内存、Spring 三级缓存）  
4. 关注新变化：Java 17/21、**虚拟线程**、G1/ZGC 调优  

```text

```

# 模拟复盘模板（错题闭环）

> **用法：** 做完任意 [模拟面试](./模拟面试入口.md) 后，把 **未过关** 的题抄进下表；  
> 每题必须：**回链主读 → 补口述 → 映射挂靠（能挂则挂）→ 第二天遮稿再测。**  
> 复制本页到本地笔记亦可。

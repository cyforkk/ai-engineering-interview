## 框架

**Q0 场景题怎么答？**  
A: 澄清→难点→分层方案→对比选型→兜底监控→项目。

## P0 秒杀 / 缓存 / 锁 / 限流 / MQ

**Q1 秒杀核心？** A: 限流削峰+Redis预扣+MQ+DB兜底。

**Q2 防超卖？** A: 原子扣减；WHERE stock>0；对账。

**Q3 防重复下单？** A: 幂等token+唯一索引。

**Q4 穿透击穿雪崩？** A: 空值布隆/互斥逻辑过期/TTL随机+HA。

**Q5 双写？** A: 先DB再删缓存；延迟双删/binlog。

**Q6 热点Key？** A: 本地缓存+互斥+打散。

**Q7 分布式锁？** A: SET NX EX+Lua；看门狗续期。

**Q8 限流？** A: 窗/桶；单机或Redis Lua。

**Q9 消息不丢？** A: 确认+持久化+成功再ACK。

**Q10 重复消费？** A: 业务幂等。

**Q11 积压？** A: 扩消费者+优消费+降级。

## P1 排查 / 订单

**Q12 CPU飙高？** A: top线程→jstack对栈。

**Q13 OOM？** A: dump→MAT引用链。

**Q14 Full GC？** A: 回收后Old降不降。

**Q15 接口变慢？** A: 链路→慢SQL/下游/锁/GC。

**Q16 订单超时？** A: 延迟队列/ZSet/扫表+状态机。

**Q17 支付幂等？** A: 单号+状态机。

**Q18 导出防OOM？** A: 流式分页写。

## P2–P3

**Q19 UV？** A: HyperLogLog。

**Q20 排行榜？** A: ZSet。

**Q21 大文件？** A: 分片秒传断点续。

**Q22 接口幂等？** A: Token/唯一键/状态机。

**Q23 雪崩防护？** A: 限流熔断降级隔离。

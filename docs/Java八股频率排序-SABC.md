# Java 高频八股 · 按面试出现频率排序（S/A/B/C）

> 综合 2024–2026 大厂/中厂面经（字节、阿里、美团、腾讯、拼多多、快手、京东、滴滴等）及 JavaGuide、牛客等常见统计。  
> **用途：先刷什么、每个点去哪学。** 面渣练嘴；卡片遮挡回忆。

> **后端四档（含 MySQL/Redis/MQ）：** [Java后端面试频率-四档.md](./Java后端面试频率-四档.md)

## 怎么用这份表

```text
1. 先死磕 S 级（覆盖约 60%+ 问题）
```

| 模块 | 状态 |

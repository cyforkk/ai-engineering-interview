## 各模块你要能答到什么程度

| 模块 | 达标 |
|------|------|
| 基础 | equals/hashCode、String、值传递、异常体系 |
| 集合 | HashMap 7/8、树化、CHM vs Hashtable |
| 并发 | volatile、线程池参数、AQS 直觉、死锁 |
| JVM | 内存区、可达性、G1、双亲委派、OOM 工具 |
| Spring | 生命周期、AOP 代理、事务失效、循环依赖 |
| MySQL | B+、最左、MVCC、EXPLAIN |
| Redis | 三连、锁、持久化、为什么快 |
| 加分 | 模式场景、CAP、消息不丢/幂等/顺序 |

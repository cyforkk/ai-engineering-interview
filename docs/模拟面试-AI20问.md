# 模拟面试 · AI / Agent 20 问

> 用法：遮住要点口述，对照勾选。  
> 主卷：[AI-Agent完整卷](./AI-Agent高频面试题与知识点.md) · 路径：[路径-AI工程.md](./路径-AI工程.md)

**及格线：** 至少 14 题能结合「原理 + 工程手段」说满 1.5 分钟。

---

## Agent 架构

### 1. 什么是 Agent？和 ChatBot 有什么本质区别？

**应覆盖：** 目标驱动多步；工具；Thought-Action-Observation；≠多聊两句。  
**链接：** [AI完整卷](./AI-Agent高频面试题与知识点.md) · [面渣](./Agent面渣级口述.md)

### 2. Agent 和 Workflow 怎么选？

**应覆盖：** 步骤固定→工作流；动态工具组合→Agent；常混合。  

### 3. 讲 ReAct，如何防止死循环？

**应覆盖：** T→A→O；max_steps、去重、超时、熔断、HITL。  

### 4. 常见 Agent 模式了解哪些？

**应覆盖：** ReAct、Plan-and-Execute、Reflexion、Multi-Agent 各一句。  

---

## RAG

### 5. 讲一遍 RAG 全流程

**应覆盖：** 加载→切→嵌→存→检索→(重排)→生成→引用。  
**链接：** [向量库](./向量库高频面试题与知识点.md) · [面渣](./向量库面渣级口述.md)

### 6. 切片大小和重叠怎么考虑？

**应覆盖：** 窗口与噪声；过大过小；overlap。  

### 7. 为什么要 Hybrid Search 和 Rerank？

**应覆盖：** 语义+关键词；粗召回精排。  

### 8. RAG 里如何降低幻觉？

**应覆盖：** 引用、拒答、阈值、评测、边界 Prompt。  

---

## 工具与记忆

### 9. Function Calling 原理？谁执行工具？

**应覆盖：** Schema→模型出意图→服务端校验执行→回灌。  

### 10. 什么是好的 Tool 描述？失败怎么处理？

**应覆盖：** 名/描述/参数；错误 Observation、熔断、HITL。  

### 11. MCP 是什么？和传统 Function Calling 差在哪？

**应覆盖：** 协议化接入工具/上下文；跨应用生态。  

### 12. Agent 记忆怎么分层？长期记忆怎么做？

**应覆盖：** 短/工作/长；向量检索；摘要压缩；权限隔离。

## 工程落地

### 13. 如何评估一个 Agent 好坏？

**应覆盖：** 成功率、步数、成本、延迟、幻觉率、接管率。  

### 14. Prompt Injection 如何防？

**应覆盖：** 不信文档指令、最小权限、校验、审批。  
**链接：** [安全](./Prompt注入与AI安全高频面试题与知识点.md)

### 15. LLMOps / 可观测你怎么做？

**应覆盖：** 版本、金标门禁、trace、token/延迟/成本。  
**链接：** [LLMOps面渣](./LLMOps与可观测面渣级口述.md)

### 16. 成本和延迟怎么优化？

**应覆盖：** 模型路由、缓存、并行工具、少步数。  

### 17. LangGraph 比线性 Chain 强在哪？

**应覆盖：** 状态机、循环、分支、HITL、可恢复。  
**链接：** [LangGraph面渣](./LangGraph与自研工作流面渣级口述.md)

### 18. Token、上下文窗口、温度怎么解释给产品听？

**应覆盖：** 计费单位、窗口预算、低温更稳。  
**链接：** [LLM面渣](./LLM基础常识面渣级口述.md)

### 19. 微调 vs RAG 怎么选？

**应覆盖：** 知识常变 RAG；风格格式可微调。

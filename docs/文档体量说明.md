# 文档体量说明

| 标记 | 含义 | 例子 |
|------|------|------|
| **完整卷** | 按超高/高/中/低写全的主笔记 | Java 基础、集合、并发、JVM、MySQL、Redis、Spring、网络、OS、Python、AI-Agent、场景题、系统设计、MQ |
| **精简 / 入口** | 提纲或索引；顶部有 ⚠️；侧栏带「精简/入口」 | AI 分册知识点、Kafka/Rocket/ES/ZK/Nginx/MyBatis/算法知识点等 |

| **冲刺 / 模拟** | 限文件清单 + 验收 | `一周冲刺-*`、`模拟面试-*` |

## 1. Agent 公式？

**A:** LLM+规划+记忆+工具+护栏。

## 2. vs 工作流？

**A:** 动态选型 vs 步骤写死。

## 3. ReAct？

**A:** Thought→Action→Observation 循环。

## 4. 防死循环 5 条？

**A:** max_steps、去重、超时、熔断、HITL。

## 5. 谁执行工具？

**A:** 服务端；模型只出调用意图。

## 6. 工具安全？

**A:** 校验、鉴权、分级、审计。

## 7. 何时不用 Agent？

**A:** 步骤固定、强合规、成本敏感。

## 8. 记忆分层？

**A:** 短时窗口；长时外存；要摘要。

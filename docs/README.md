## 2. 推荐用法

```text
① 打开「知识点」只刷 P0 表（15～25min）
② 打开「面渣级口述」对 P0 题录音练（每题 2～4min）
③ 用「追问与手写」模拟二面
④ 有余力再刷 P1 / P2
```

**时间紧：** 只练序 1～6 的面渣 P0 长口述。

## 3. 各专题面渣 P0 题眼（先练这些）

| 专题 | 面渣优先题 |
|------|------------|
| Java | equals/hashCode、String 不可变、**HashMap 长答**、ArrayList |
| 并发 | **线程池长答**、volatile、synchronized vs Lock、CAS |
| JVM | 内存区域、GC、双亲委派、**OOM/FullGC/CPU 场景** |
| MySQL | B+索引、最左、隔离级别、**MVCC**、慢 SQL |
| Redis | 为什么快、**穿透击穿雪崩**、**分布式锁** |
| Spring | IoC/AOP、**事务失效**、**三级缓存** |
| MQ | 为何用、不丢、幂等、积压 |
| 网络 | 三次握手、四次挥手、HTTPS |
| OS | 进程线程、epoll |
| Agent | 定义对比、ReAct 防循环、Tool 调用 |

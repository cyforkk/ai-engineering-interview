## 1. 什么是 Token？上下文窗口？

- **Token**：模型计费与长度的基本单位（子词片），≠ 字数。
- 中文往往更耗 token。
- **上下文窗口**：一次请求能塞下的上限（系统提示 + 历史 + 检索 + 问题 + 预留输出）。
- 超窗：摘要历史、减少检索条数/长度、截断旧轮。

---

## 2. 温度（temperature）？

- 控制生成随机性。
- **低**：更稳，适合抽字段、客服、工具参数。
- **高**：更发散，易跑偏。

---

## 3. 什么是幻觉？怎么压？

- 流畅但错误/捏造的内容。
- 原因：生成式预测下一 token，不是查权威库。
- 手段：RAG 引用、低相关拒答、校验、金标评测；不承诺永不幻觉。

---

## 4. 预训练 / SFT / RLHF？

- **预训练**：海量文本学语言与知识。
- **SFT**：指令数据学会听人话。
- **RLHF/偏好对齐**：更有用、更安全、更符合偏好。
- 知识常变 → 优先 **RAG**；风格/格式稳定 → 可考虑微调。

---

## 5. 云 API vs 私有化？

- 云：上线快、运维轻，数据出域风险。
- 私有：可控，要 GPU 与运维。
- 常混合 + 统一网关。

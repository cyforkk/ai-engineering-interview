## 1. Agent 是什么？

- 目标驱动：多步推理 + 调工具 + 观察结果 + 再决策，直到完成或停止。
- 公式：`Agent ≈ LLM + 规划 + 记忆 + 工具 + 护栏`。

---

## 2. Chat / RAG / 工作流 / Agent 区别？

- **Chat**：生成回复。
- **RAG**：先检索再生成。
- **工作流**：步骤人写死，可控。
- **Agent**：步骤模型动态选，灵活但贵、难控。
- 步骤固定、强合规、成本敏感 → 优先工作流，不必上 Agent。

---

## 3. ReAct？

- Thought → Action → Observation 循环 → Final Answer。
- 风险：死循环、步数/费用爆炸。
- **护栏**：max_steps、重复检测、超时、工具错误熔断、危险操作 HITL。

---

## 4. 工具调用安全？

- 模型只输出调用意图；**服务端执行**。
- 参数校验、鉴权、最小权限、审计日志、写操作审批。

---

## 5. 记忆？

- 短时：对话窗口；长时：向量/DB；过长要摘要。

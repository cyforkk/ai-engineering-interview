## 1. 为何用 MQ？

**A:** 解耦、削峰、异步。

## 2. 至少一次？

**A:** 可能重复→消费幂等。

## 3. 顺序消息？

**A:** 分区/队列键保序。

## 4. 积压？

**A:** 扩消费者、降耗时、限速生产。

## 5. Kafka 分区？

**A:** 并行度单位；键决定分区。

## 6. RocketMQ 事务消息？

**A:** 半消息+本地事务回查。

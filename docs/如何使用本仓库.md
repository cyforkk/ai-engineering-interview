# 如何使用本仓库（必读）

---

## 三分法（先记住这个）

| 类型 | 文件名 | 干什么 | 什么时候用 |
|------|--------|--------|------------|
| **知识点详解** | `…知识点.md` | **讲清楚、讲所以然**（是什么 / 为什么 / 怎么工作 / 边界） | 第一次学、被追问「为什么」 |
| **卡片速记** | `…卡片速记.md` | **极简回忆**（Q→A 或一句话要点） | 通勤、考前、遮挡自测 |
| **面渣口述** | `…面渣级口述.md` | **怎么开口说**（2～4 分钟口语稿） | 练嘴、录音、模拟面试 |

**面渣不改定位：只练「说」；原理看详解；背要点看卡片。**

```text
不懂原理  → 知识点详解（所以然）
要开口说  → 面渣口述（主口语教材，保持不动）
要快速回忆 → 卡片速记（极简）
```

---

## 推荐读法（每天一套）

```text
① 路径表找到今天一步
② 知识点详解：把「为什么」读通（可做笔记）
③ 面渣口述：通读 → 对着说 2 分钟 → 遮稿再说
④ 卡片速记：遮住答案，只看 Q 回忆
⑤ 追问三连 / 自测
```

| 错误用法 | 后果 |
|----------|------|
| 只刷卡片关键词 | 会词不会讲所以然 |
| 只读详解不练嘴 | 懂但说不利索 |
| 只背面渣不求甚解 | 换个问法就崩 |

## 练习标准

| 程度 | 标准 |
|------|------|
| 不及格 | 只会甩名词 |
| 及格 | 卡片能回忆 + 面渣能看稿说 2 分钟 |
| 良好 | 详解里的「为什么」能讲清 + 遮稿口述 + 接追问 |
| 优秀 | 接到自己项目举例（STAR） |

---

## 一句话

> **详解求懂，面渣求说，卡片求记。三者分工，不要混成一种文档。**

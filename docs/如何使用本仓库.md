# 如何使用本仓库（必读）

---

## 三分法

| 类型 | 干什么 | 风格 |
|------|--------|------|
| **知识点详解** | 把原理**讲懂** | **大白话**：先一句话 → 生活例子 → 慢慢讲 → 易混 |
| **卡片速记** | 考前**回忆** | 极短 Q→A |
| **面渣口述** | 面试**怎么说** | 完整口语稿（不动） |

```text
看不懂  → 读详解（生活例子）
要开口  → 练面渣
要背点  → 刷卡片
```

---

## 详解怎么读才不累

每个考点尽量按这个顺序：

1. **先记一句** — 抓住中心  
2. **生活例子** — 建立感觉  
3. **慢慢讲** — 机制  
4. **易混/易错** — 防踩坑  
5. **自测** — 用人话答出来  

还是懵？不要硬啃术语，换例子想；再去面渣听「怎么说」。

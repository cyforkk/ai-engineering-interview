# 如何使用本仓库（必读）

---

## 1. 只记三件事

| 文档 | 干什么 |
|------|--------|
| **知识点** | 过题、记结构（笔记体） |
| **面渣口述** | **练怎么说**（2～4 分钟） |
| **卡片** | 考前遮挡回忆 |

```text
不会说 = 没练面渣，不是你笨
```

---

## 2. 唯一正确入口

```text
首页 → 选路径 A / B / C → 按表从上往下
```

- **不要**在侧栏长列表里乱点  
- 频率表、完整卷是路径里的链接，不是起点  

| 岗位 | 路径 |

| AI / Agent | [路径-AI工程.md](./路径-AI工程.md) |
| Java 后端 | [路径-Java后端.md](./路径-Java后端.md) |

Java 冲刺 P0：[四档主线](./Java后端面试频率-四档.md)

---

## 3. 每天一套（可复制）

```text
① 路径当前步打开「知识点」过 3～5 题
② 打开「面渣」通读 → 对着说 2 分钟（可录音）
③ 遮稿再说一遍
④ 卡片遮 A 回忆
⑤ 周末做「模拟面试」一页
```

| 程度 | 标准 |
|------|------|
| 不及格 | 只会甩名词 |
| 及格 | 面渣结构说满 2 分钟，偶看稿 |
| 良好 | 遮稿 + 接 2～3 个追问 |
| 优秀 | 接到自己项目举例 |

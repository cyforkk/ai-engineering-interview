# 如何使用本仓库（必读）

---

## 三分法

| 类型 | 干什么 | 长什么样 |
|------|--------|----------|
| **知识点** | 记考点、过结构 | **面试笔记体**：`1. 问题？` + 分层 `●/○` 要点 |
| **卡片速记** | 考前遮挡回忆 | 极短 Q→A |
| **面渣口述** | 练「怎么说」 | 2～4 分钟口语稿（不动） |

```text
背结构 / 过题  → 知识点（笔记体）
练开口        → 面渣
遮挡回忆      → 卡片
```

---

## 知识点示例风格

```text
1. == 和 equals() 的区别？
● ==：
  ○ 基本类型：比值
  ○ 引用类型：比地址
● equals()：
  ○ 默认：比地址
  ○ 重写后：比内容
```

要展开成完整面试口语 → 打开同名 **面渣**。

---

## 每天一套

```text
知识点过题 → 面渣说 2 分钟 → 卡片回忆 → 追问
```

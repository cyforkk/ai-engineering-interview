## 3. 每天一套（可复制）

```text
① 路径当前步打开「完整卷或面渣」（勿点精简当主读）
② 面渣对着说 2 分钟（可录音）
③ 遮稿再说一遍
④ 卡片遮 A 回忆
⑤ 把 1 个点写进项目映射表
⑥ 周末做「模拟面试入口」一页
```

| 程度 | 标准 |
|------|------|
| 不及格 | 只会甩名词 |
| 及格 | 面渣结构说满 2 分钟，偶看稿 |
| 良好 | 遮稿 + 接 2～3 个追问 |
| 优秀 | 接到自己项目举例（见映射样例） |

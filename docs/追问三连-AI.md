## 场景三连

### 主问：企业知识库问答安全吗？

| # | 追问 | 要点 |
|---|------|------|
| 1 | 员工 A 能否搜到 B 的文档？ | 检索强制 ACL |

| # | 追问 | 要点 |

| 2 | 延迟和费用？ | 多轮 LLM |
| 3 | 失败怎么降级？ | 回工作流/人工 |

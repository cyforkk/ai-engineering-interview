# 项目映射模板（简历 → 面试话术）

> **目的：** 把简历项目翻译成面试官能追问的「场景题答案」。  
> 每个项目填 1 张表；准备 **1 主项目 + 1 备选** 即可。  
> 不会填时，先看 [场景题完整卷](./场景题高频面试题与知识点.md) / [STAR 范例](./项目STAR范例-Java.md)。

---

## 使用方法

1. 复制下面「空白表」到自己的笔记  
2. 用简历真实经历填（没有就写「课程/练手项目」并诚实说明）  
3. 每个「可挂靠场景」准备 2 分钟口述  
4. 用模拟面试题抽查

> **目的：** 降低选择疲劳。按岗位打开清单，**不要**在侧栏检索区乱点。

## 怎么用

1. 选一个岗位区块  
2. 只打开「必碰」列表（约 10～15 个文件）  
3. 验收走 [模拟入口](./模拟面试入口.md) + [复盘模板](./模拟复盘模板.md)  
4. 有余力再开「可选」

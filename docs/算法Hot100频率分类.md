## 一、哈希表

**本质**：把「在集合里反复查找」从 `O(n)` 降到 `O(1)`。  
**信号词**：两数配对、计数、去重、连续序列、前缀和次数。

### 模板

```python
# 值 -> 下标 / 次数
index = {}
for i, x in enumerate(nums):
    need = target - x
    if need in index:
        return [index[need], i]
    index[x] = i
```

### 高频题（频率高 → 低，题号升序辅助）

| 频率档 | 题号 | 题目 | 核心一点 |
|--------|------|------|----------|
| **S** | 1 | 两数之和 | 边扫边查 `target - x` |
| **S** | 49 | 字母异位词分组 | 排序串或 26 计数作 key |
| **A** | 128 | 最长连续序列 | 只从序列起点扩展 |
| **A** | 560 | 和为 K 的子数组 | 前缀和 + 哈希计次数 |
| **B** | 347 | 前 K 个高频元素 | 计数 + 堆 / 桶排序 |

**128 要点**：先入 `set`，仅当 `x-1` 不存在时才从 `x` 往右数长度，保证均摊 `O(n)`。

**560 要点**：`pre - k` 在哈希表中出现过，说明存在以当前为右端点、和为 `k` 的子数组；初始化 `count[0] = 1`。

```python
count = {0: 1}
pre = ans = 0
for x in nums:
    pre += x
    ans += count.get(pre - k, 0)
    count[pre] = count.get(pre, 0) + 1
```

## 二、双指针

**本质**：有序或可排序后，用左右边界压缩搜索空间。  
**信号词**：有序数组配对、对撞、快慢、原地分区。

### 模板（对撞）

```python
left, right = 0, n - 1
while left < right:
    s = nums[left] + nums[right]
    if s == target:
        # 记录 / 去重后收缩
        left += 1
        right -= 1
    elif s < target:
        left += 1
    else:
        right -= 1
```

### 高频题

| 频率档 | 题号 | 题目 | 核心一点 |
|--------|------|------|----------|
| **S** | 15 | 三数之和 | 排序 + 固定 i + 对撞 + 去重 |
| **S** | 11 | 盛最多水的容器 | 移动较短边 |
| **A** | 283 | 移动零 | 慢指针写非零 |
| **A** | 42 | 接雨水 | 左右最大高度取 min |
| **B** | 75 | 颜色分类 | 三指针分区 |

**15 要点**：`i` 去重；命中后 `left/right` 都要跳过相同值。

**11 要点**：宽度只减不增，只能靠抬高「短板」碰更大答案。

```python
# 11 盛水
L, R, ans = 0, len(height) - 1, 0
while L < R:
    ans = max(ans, (R - L) * min(height[L], height[R]))
    if height[L] < height[R]:
        L += 1
    else:
        R -= 1
```

## 三、滑动窗口

**本质**：维护满足条件的区间 `[left, right]`，右端扩张、左端收缩。  
**信号词**：最长/最短子串、覆盖、异位词窗口。

### 模板

```python
left = 0
window = {}  # 或计数数组
for right, x in enumerate(s):
    # 1. 右端加入
    window[x] = window.get(x, 0) + 1
    # 2. 不合法则缩左
    while not_valid(window):
        window[s[left]] -= 1
        left += 1
    # 3. 用当前窗口更新答案（最长 or 最短）
    update_ans(left, right)
```

### 高频题

| 频率档 | 题号 | 题目 | 核心一点 |
|--------|------|------|----------|
| **S** | 3 | 无重复字符的最长子串 | 窗口内字符不重复 |
| **A** | 438 | 找到字符串中所有字母异位词 | 定长窗口 + 计数相等 |
| **A** | 76 | 最小覆盖子串 | 覆盖后尽量缩左 |
| **B** | 239 | 滑动窗口最大值 | 单调递减队列存下标 |

**3 示例**：

```python
last = {}  # 字符最近下标
left = ans = 0
for right, c in enumerate(s):
    if c in last and last[c] >= left:
        left = last[c] + 1
    last[c] = right
    ans = max(ans, right - left + 1)
```

**239 要点**：队首是窗口最大；弹出越界下标；从队尾弹出所有 `<=` 新值的下标，保持递减。

## 五、二叉树

**本质**：递归定义子问题；遍历决定信息收集顺序。  
**信号词**：深度、路径、构造、层序、最近公共祖先、BST。

### DFS / BFS 骨架

```python
# DFS
def dfs(node):
    if not node:
        return base
    L, R = dfs(node.left), dfs(node.right)
    # 用 L、R、node 更新全局或返回值
    return combine(L, R, node)

# BFS 层序
q = deque([root])
while q:
    level = []
    for _ in range(len(q)):
        node = q.popleft()
        level.append(node.val)
        if node.left:  q.append(node.left)
        if node.right: q.append(node.right)
```

## 六、动态规划

**本质**：大问题拆成有最优子结构的小问题，用状态转移避免重复计算。  
**信号词**：最大/最小、方案数、能否、路径、背包、序列。

### 通用步骤

1. **定义 `dp[i]`（或 `dp[i][j]`）含义**  
2. **写转移与初始值**  
3. **定遍历顺序**（无后效）  
4. **答案从哪取**

```python
# 一维典型：最大子数组和
# f[i] = 以 i 结尾的最大和
f = nums[0]
ans = nums[0]
for i in range(1, n):
    f = max(nums[i], f + nums[i])
    ans = max(ans, f)
```

### 高频题

| 频率档 | 题号 | 题目 | 状态直觉 |
|--------|------|------|----------|
| **S** | 70 | 爬楼梯 | `f[i] = f[i-1] + f[i-2]` |
| **S** | 53 | 最大子数组和 | 延续 or 重开 |
| **S** | 121 | 买卖股票的最佳时机 | 记历史最低价 |
| **S** | 322 | 零钱兑换 | 完全背包最少枚举 |
| **A** | 300 | 最长递增子序列 | `O(n^2)` DP / 贪心+二分 |
| **A** | 139 | 单词拆分 | 前缀可拆则扩展 |
| **A** | 152 | 乘积最大子数组 | 同时维护最大最小 |
| **A** | 62 | 不同路径 | 网格路径计数 |
| **B** | 72 | 编辑距离 | 二维字符串 DP |
| **B** | 416 | 分割等和子集 | 0-1 背包 |

**322 示例**：

```python
dp = [0] + [inf] * amount
for a in range(1, amount + 1):
    for c in coins:
        if c <= a:
            dp[a] = min(dp[a], dp[a - c] + 1)
return dp[amount] if dp[amount] != inf else -1
```

**面试优先掌握**：一维线性 DP、网格、背包（0-1 / 完全）、买卖股票一类状态机。

## 七、回溯

**本质**：在决策树上下 DFS，选 / 不选或枚举位置，回溯撤销选择。  
**信号词**：全排列、子集、组合、括号、棋盘搜索。

### 模板

```python
def backtrack(path, choices):
    if is_solution(path):
        ans.append(path[:])
        return
    for i, c in enumerate(choices):
        if used[i]:
            continue
        # 可选：剪枝 / 去重（排序后同层跳过相同）
        used[i] = True
        path.append(c)
        backtrack(path, choices)
        path.pop()
        used[i] = False
```

### 高频题

| 频率档 | 题号 | 题目 | 核心一点 |
|--------|------|------|----------|
| **S** | 46 | 全排列 | 用 `used` 标记 |
| **S** | 78 | 子集 | 每位选或不选 / 顺序构造 |
| **A** | 39 | 组合总和 | 可重复用，下一轮从 i 开始 |
| **A** | 22 | 括号生成 | `left < n`，`right < left` |
| **A** | 17 | 电话号码的字母组合 | 数字映射字符 DFS |
| **B** | 79 | 单词搜索 | 网格四向 + 回溯标记 |

**去重口诀**（组合/子集 II 类）：先排序，同层若 `nums[i] == nums[i-1]` 且前一个未选，则跳过。

## 八、栈与单调栈

**本质**：后进先出处理嵌套；单调栈找「下一个更大/更小」。  
**信号词**：括号匹配、解码、每日温度、柱状图、接雨水。

### 单调栈（下一个更大）

```python
stack = []  # 存下标，对应值单调递减
ans = [0] * n
for i, x in enumerate(nums):
    while stack and nums[stack[-1]] < x:
        j = stack.pop()
        ans[j] = i - j  # 或 nums[i]
    stack.append(i)
```

| **S** | 20 | 有效的括号 | 栈匹配左右括号 |
| **A** | 739 | 每日温度 | 单调递减栈 |
| **A** | 155 | 最小栈 | 辅栈同步最小值 |
| **A** | 394 | 字符串解码 | 数字栈 + 字符串栈 |
| **B** | 84 | 柱状图中最大的矩形 | 左右第一个更小柱 |

**20 要点**：遇右括号则栈顶必须是对应左括号；结束栈必须空。

## 九、二分查找

**本质**：在有序或「答案具有单调性」的空间上折半。  
**信号词**：有序数组、旋转数组、边界（第一个/最后一个）、最小值最大化。

### 模板（左闭右开求下界）

```python
left, right = 0, n  # [left, right)
while left < right:
    mid = (left + right) // 2
    if nums[mid] < target:
        left = mid + 1
    else:
        right = mid
# left 为第一个 >= target 的位置
```

### 高频题

| 频率档 | 题号 | 题目 | 核心一点 |
|--------|------|------|----------|
| **S** | 33 | 搜索旋转排序数组 | 半边有序，判断 target 落点 |
| **A** | 34 | 在排序数组中查找元素的第一个和最后一个位置 | 两次边界二分 |
| **A** | 4 | 寻找两个正序数组的中位数 | 划分两数组使左右平衡 |
| **B** | 240 | 搜索二维矩阵 II | 从右上角缩小行列 |

**写二分先定**：搜索空间、`mid` 归属哪半、循环不变量、退出后校验。

**56**：

```python
intervals.sort(key=lambda x: x[0])
ans = []
for a, b in intervals:
    if not ans or ans[-1][1] < a:
        ans.append([a, b])
    else:

```

**1143 模板**：

```python
# dp[i][j] = LCS(text1[:i], text2[:j])
m, n = len(a), len(b)
dp = [[0] * (n + 1) for _ in range(m + 1)]
for i in range(1, m + 1):
    for j in range(1, n + 1):
        if a[i - 1] == b[j - 1]:
            dp[i][j] = dp[i - 1][j - 1] + 1
        else:
            dp[i][j] = max(dp[i - 1][j], dp[i][j - 1])
return dp[m][n]
```

**518 注意**：外层物品、内层容量 → 组合数；反过来容易变成排列数。

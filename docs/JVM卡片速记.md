---

## 1. 运行时内存分区？

**A:** 私有：PC/栈/本地方法栈；共享：堆、方法区/元空间；另有直接内存。

## 2. 堆分代？

**A:** 新生代 Eden+S0/S1；老年代；朝生夕死 vs 长寿。

## 3. GC Roots 例子？

**A:** 栈引用、静态变量、常量、JNI、锁对象等。

## 4. 如何判断垃圾？

**A:** 可达性分析，不是引用计数（循环引用）。

## 5. Minor vs Full GC？

**A:** Minor 收新生代；Full 范围大停顿长。

## 6. G1 一句话？

**A:** Region 堆，优先回收收益高的区，可控停顿目标。

## 7. 双亲委派？

**A:** 先父后子加载，保证核心类唯一。

## 8. 堆 OOM 第一步？

**A:** dump + MAT 看占用与引用链。

## 9. CPU 飙高？

**A:** top 找线程 → jstack 对栈。

## 10. 容器无 Java OOM 被杀？

**A:** RSS 超 cgroup，堆外也算。

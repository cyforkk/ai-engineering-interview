## 1. 进程 vs 线程？

**A:** 资源分配 vs 调度；线程共享堆要同步。

## 2. 用户态/内核态？

**A:** 权限隔离；syscall 切换有成本。

## 3. 上下文切换？

**A:** 保存/恢复现场；过多损吞吐。

## 4. epoll 一句话？

**A:** 多路复用，就绪通知，适合海量连接。

## 5. 死锁四条件？

**A:** 互斥、持有等待、不可抢占、环路。

## 6. 虚拟内存？

**A:** 页表映射；缺页走磁盘慢。

## 7. 零拷贝？

**A:** 少用户/内核数据拷贝。

## 8. Java 线程和 OS？

**A:** 通常 1:1 映射到内核线程（HotSpot）。
